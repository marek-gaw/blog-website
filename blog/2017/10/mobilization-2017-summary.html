Mobilization 2017 and TDD Workshop - summary

Posted on nie 29 października 2017 in General • 2 min read

Mobilization is probably the biggest IT conference which happens in Łódź, each year since 2011. I attend this event each year since 2015 edition, so it was my third time, and I was pretty sure what I can expect.

Presentations I attended:

  1. How to combine Swift/Kotlin with C/C++ by Michał Kowalczyk

    This one was interesting, because it involved C++. Michał showed, how to actually use C++ on mobile: his idea is to create reusable, platform-agnostic backend of application, and native frontend, for Android or iOS. This idea is worth considering, if your application has to be created and maintained for all (Android, iOS, Windows) platforms.

  2. Android Things 101 by Raul Portales

  3. Using Android Things to detect & exterminate Reptilians by Gautier Mechling

    Both of those two presentations were more or less about the same: introduction to Android Things. Those presentation were not too deep in details, but gave me pretty good insight in Android Things, enough to draw my attention.

This time, before the main event, I attended workshop, made by guys from EL Passion.

From my point of view, this workshop was mainly about TDD in practice. Eventually, it was far more demanding: the main programming language was Kotlin, and we were using extensively RxJava. Obviously, those are not my main technologies, as Android is not, but I was going there to familiarize myself with Test Driven Development. Also, this was an opportunity to get to know Kotlin, which looks like very interesting replacement for Java. I think I need to go deeper in this topic, as it seems, that for Android, this will be main and leading technology.

For those, who would like to know the agenda:Workshop announcement. For those, who would like to attend this workshop, you can do this for free Workshop - I strongly recommend it!

To sum up: the conference gave me an opportunity to learn something new, despite of the fact, that mobile is not my main topic of interest. Also, workshops, which were organized for the first time, is a great idea.
