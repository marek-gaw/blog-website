Robotics Toolbox and V-REP

Posted on nie 07 lutego 2016 in Robotics • 1 min read

Recently I finally managed to fix communication between Matlab and V-REP.

Short introduction:
  • Robotics Toolbox : Peter Corke created great tool: RTB, which is toolbox for Matlab.
  • V-REP : V-REP by Coppelia Robotics is simulation environment.

RTB has an option to communicate with V-REP as visualization tool.

I tried to run vrepdemo.m form RTB but I have encountered a problem with running it. My problem and solution was described on Google Groups: Robotics and Machine Vision Toolboxes
