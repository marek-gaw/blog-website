Quick Tip: How to setup rosbridge

Posted on Å›ro 24 sierpnia 2016 in Robotics • 1 min read

For last 3 days I tried to establish communication between ROS node and non-ROS application - in my case it was Python app. I was able to publish data to topic, but I was not able to receive any, despite of the fact, that rosbridge server accepted my request to subscribe to topic.

Finally, I found the solution in ![this topic on ROS Answers]. Basically, the problem is that, during rosbridge installation I have installed bson packet with pip. But I HAVE TO HAVE version from MongoDB package! So, basically…

pip uninstall bson
pip -m install pymongo

…fixed my problem.

![this topic on ROS Answers]: https://answers.ros.org
