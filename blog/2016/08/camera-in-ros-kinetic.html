Camera in ROS Kinetic

Posted on sob 20 sierpnia 2016 in Robotics • 1 min read

How to make camera work in ROS Kinetic?

sudo apt install ros-kientic-image-pipeline

This will install package image_pipeline which contains useful things. What we actually need is package image_view.

Second thing is usb_cam. Obviously, this one is not included in Kinetic… So, we have to build it on our own.

So, lets compile usb_cam!

git clone https://github.com/bosch-ros-pkg/usb_cam.git
catkin_make
catkin_make -install
roslaunch usb_cam usb_cam-test.launch
