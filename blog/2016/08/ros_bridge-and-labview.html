ros_bridge and LabVIEW

Posted on czw 04 sierpnia 2016 in Robotics • 1 min read

For now, rosbridge is not available as .deb package, so I need to compile it by myself.

Rosbridge can be cloned from GitHub:

git clone https://github.com/RobotWebTools/rosbridge_suite.git

There is a dependency with package rosauth - this package also needs to be compiled manually.

git clone https://github.com/GT-RAIL/rosauth.git

What I found interesting, there is a dependency with Python package bson

pip2 install bson

I had to use pip2 explicitly as I have Python3 installed among Python2 on my laptop.

Basically, usual catkin build is sufficient to build both packages and run rosbridge server.

Update¶

Since December 2017 rosbridge can be installed with apt as package:

sudo apt install ros-kinetic-rosbridge
