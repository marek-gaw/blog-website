Quick tip: rqt plugins in ROS Kinetic

Posted on pon 15 sierpnia 2016 in General • 1 min read

I just tried to use rqt in ROS Kinetic and I found, that only one plugin is installed by default: Container.

I went trough some pages, and found some helpful info.

First of all, install additional packges with plugins:

sudo apt install ros-kinetic-rqt*

If rqt is aware of installed plugins, you will see them on the list:

rqt --list-plugins

If this list is empty, or there is only few of plugins, force rqt to discover:

rqt --force-discover

This will also run rqt, and all installed plugins should be available.
